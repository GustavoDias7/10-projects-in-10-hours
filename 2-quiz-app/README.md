# Quiz App
 
## O que eu aprendi?
 
Nos meus programas anteriores em JS, eu tentava separar o programa em partes. Por exemplo: uma parte com as funções, uma com os seletores dos elementos HTML (DOM), outra com as variáveis mais importantes e por fim uma com os eventos interativos.
 
Parece mais fácil de desenvolver, ler e entender o código, quando ele segue uma sequência lógica, como num enunciado de problema de algoritmo.
 
A grosso modo, a lógica desse projeto é:
- Criar as informações das fases.
- Selecionar e armazenar os elementos HTML utilizados.
- Criar as funções para executar a lógica interna.
- Adicionar o evento de clique do botão principal.
